# 🌤️ Weather App

A simple weather app that fetches real-time weather data for any city.

## What it does
- Type in any city name
- Hit Search
- instantly shows temperature and weather condition

## What I actually learned building this
- fetch API and how to work with real live data
- async/await in JavaScript
- Reading and using API documentation
- Working with JSON responses
- Connecting a frontend to a real world API

## Built with
- HTML
- CSS
- Vanilla JavaScript
- OpenWeatherMap API
