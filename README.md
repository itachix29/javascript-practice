# ✅ My First To-Do List App

Yeah, it's a to-do list. Everyone builds one — but this one's mine.

Built this while learning JavaScript for the first time. 
Took me a while to figure out why things weren't showing up 
correctly (CSS Grid humbled me), but got there in the end.

## What it does
- Type in a task and a due date
- Hit Add — it shows up instantly
- Hit Delete — it's gone
- That's it. It's a to-do list.

## What I actually learned building this
- How JavaScript arrays and objects work together
- Manipulating the DOM (making JS talk to HTML)
- CSS Grid — fought with it, won eventually
- Writing dynamic HTML using template literals
- Why wrapping divs in a grid is a bad idea (learned the hard way)

## Built with
- HTML
- CSS  
- Vanilla JavaScript (no frameworks, no libraries, just vibes)
